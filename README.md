# Chat Application with Angular and Socket.io

## Development server

To start a local development server, run:

```bash
ng serve
```

## Backend link
To see the backend side of this application click this [link](https://github.com/Movgto/chat-backend)


## Live site
You can see the live site of the application [here](https://chat-maro.netlify.app/)
> The backend is hosted in a free instance render web service, so the server might take a few moments to be functional.
